# League Standings by Gameweek Webapp

## Information
Streamlit webapp that generates Seasonal League Standings overviews with Bezier Curves.

Supported leagues:
* `Premier League`
* `Serie A`
* `Ligue 1`
* `Bundesliga`
* `Eredivisie`


## Data
Data scraped from [worldfootbal.net](https://www.worldfootball.net/).
